# endgameAssembly

### Installation

1. Clone the repository
```bash
git clone https://github.com/kancherish/endgameAssembly.git
cd endgameAssembly
```

2. Install dependencies
```bash
npm install
```

2. RUN
```bash
npm run dev
```
